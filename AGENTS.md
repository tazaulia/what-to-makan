# Agent Guidelines

## Setup
- Install dependencies using `npm i` or run `./setup.sh`.
- Ensure Bun is installed from [bun.sh](https://bun.sh).
- Copy `.env.example` to `.env` and fill in credentials.

## Testing
- Run `bun test` before committing.
- Optionally run `npm run lint`.

## Commit messages
- Use concise, present-tense messages describing the change.
